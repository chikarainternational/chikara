data.js の構造 data.js Structure

すべてのコンテンツは js/data.js の DATA オブジェクト1箇所で管理されています。ここを編集するだけで day.html / list.html / month.html の全ページに即反映されます。

All content is managed in a single DATA object in js/data.js. Editing it instantly updates all pages: day.html, list.html, and month.html.

var DATA = {
  months: [
    {
      no: 1,                          // 月番号
      name: "Month 1",                // 月名
      icon: "📘",                     // アイコン絵文字
      iconBg: "m1-bg",               // 背景色クラス
      desc: "I told you / Come on…", // 月の概要（トップページ表示）
      meta: "レッスン 4回 ｜ …",      // メタ情報（トップページ表示）
      lessons: [
        {
          no: 1,                      // レッスン番号
          day: "Day 1〜4",            // Day 表示テキスト
          title: "I told you / …",   // レッスンタイトル（月別ページ表示）
          listen: [
            {
              en:      "Did you",             // 英語フレーズ（読み上げ対象）
              kana:    "ディジュ",             // カタカナ発音
              note:    "d が弱くなる",         // 発音変化の説明（任意）
              example: "Did you get it?"      // 例文（任意）
            }
          ],
          learn: [
            {
              phrase:   "I told you",         // 英語フレーズ
              jp:       "昨日言ったでしょ",    // 日本語訳
              desc:     "tell + 人 で…",      // 文法説明（任意）
              examples: [
                { en: "I told you last night.", jp: "" }
              ]
            }
          ]
        }
      ]
    }
  ]
}

コンテンツの編集 Editing Content

コンテンツの正本は document/Extra English 学習ポイント・リスニングポイント.pdf です。編集の際は必ずPDFと照合してください。

The canonical content source is document/Extra English 学習ポイント・リスニングポイント.pdf. Always cross-reference this PDF when editing.

リスニングポイントの編集 Editing Listening Points

対象レッスンの listen 配列を編集します。

Edit the listen array for the target lesson.

listen: [
  { en: "Did you",   kana: "ディジュ",   note: "d が弱くなる", example: "Did you get it?" },
  { en: "I told you", kana: "アイ トウジュ", note: "",           example: "" }
]
note と example は省略可です。不要な場合は空文字列 "" を入れてください。
note and example are optional. Use empty strings "" if not needed.

学習ポイントの編集 Editing Learning Points

対象レッスンの learn 配列を編集します。

Edit the learn array for the target lesson.

learn: [
  {
    phrase: "I told you",
    jp: "昨日言ったでしょ",
    desc: "tell + 人 で「人に言う」という意味。",
    examples: [
      { en: "I told you last night.", jp: "昨夜言ったじゃない。" }
    ]
  },
  {
    phrase: "Don't + 動詞",
    jp: "〜しないで",
    desc: "",
    examples: [
      { en: "Don't cry.", jp: "" },
      { en: "Don't worry.", jp: "" }
    ]
  }
]

desc が空かつ examples が空配列の場合、アコーディオンではなく通常リスト表示になります。

If both desc is empty and examples is an empty array, the item renders as a plain list row instead of an accordion.

レッスンの追加 Adding a Lesson

js/data.js だけを編集します。HTMLファイルの変更は一切不要です。

Edit js/data.js only. No HTML changes are needed.

既存の月にレッスンを追加する Add a lesson to an existing month

対象の月の lessons 配列に新しいオブジェクトを追加します。no は連番で付けてください。

Append a new object to the lessons array of the target month. Assign no as the next sequential number.

lessons: [
  { no: 1, day: "Day 1〜4",  ... },
  { no: 2, day: "Day 8〜11", ... },
  // ↓ 追加 / Add
  { no: 3, day: "Day 15〜18", title: "...", listen: [...], learn: [...] }
]

新しい月（Month 4 以降）を追加する Add a new month (Month 4+)

DATA.months 配列に新しい月オブジェクトを追加します。合わせて index.html に月カードを追加してください（index.html のみ静的）。

Append a new month object to the DATA.months array. Also add a month card to index.html (the only static page).

months: [
  { no: 1, ... },
  { no: 2, ... },
  { no: 3, ... },
  // ↓ 追加 / Add
  {
    no: 4,
    name: "Month 4",
    icon: "📒",
    iconBg: "m4-bg",
    desc: "フレーズの概要...",
    meta: "レッスン X回 ｜ ...",
    lessons: [
      { no: 1, day: "Day X〜Y", title: "...", listen: [...], learn: [...] }
    ]
  }
]
iconBg に対応する背景色クラスは style.css に追加が必要です（例: .m4-bg { background: #...; }）。
The iconBg background color class must be added to style.css (e.g. .m4-bg { background: #...; }).

共通機能（script.js） Shared Features (script.js)

js/script.js は全ページで読み込まれ、3つの機能を提供します。このファイルを直接編集する必要は通常ありません。

js/script.js is loaded on all pages and provides three features. You typically do not need to edit this file.

① 右クリック禁止 ① Right-click disabled

コンテキストメニューを全ページで無効化しています。意図的な設定です。

The context menu is disabled on all pages. This is intentional.

document.addEventListener("contextmenu", function(e) { e.preventDefault(); });

② 検索オーバーレイ ② Search overlay

ヘッダー右端に🔍ボタンを自動挿入します。クリックで全コンテンツ横断検索が可能です。

A 🔍 button is automatically added to the right side of the header. Click it to search across all content.

  • 学習ポイント（フレーズ・日本語訳・文法説明・例文）を横断検索
  • リスニングポイント（英語・カタカナ・発音ノート）を横断検索
  • 結果クリックで該当の day.html?m=X&l=Y へ遷移
  • Searches learning points (phrase, translation, grammar notes, examples)
  • Searches listening points (English, kana, pronunciation notes)
  • Clicking a result navigates to day.html?m=X&l=Y

③ 音声読み上げ（TTS） ③ Text-to-Speech (TTS)

リスニングポイントの英語フレーズ横にスピーカーボタンを自動挿入します。クリックで Web Speech API によるブラウザ音声読み上げが実行されます（lang: en-US / 速度 0.85）。読み上げ中はボタンが青くパルス点滅します。

A speaker button is automatically placed next to each English phrase in the listening points. Clicking it triggers browser text-to-speech via the Web Speech API (lang: en-US, rate 0.85). The button pulses blue while speaking.

ブラウザBrowser 対応状況Support
Chrome / Edge○ 完全対応○ Full support
Safari (iOS / macOS)○ 対応○ Supported
Firefox△ 音声が少ない場合あり△ Limited voice availability
Web Speech API 非対応ブラウザではボタンが表示されません。コンテンツの閲覧には影響ありません。
On unsupported browsers, the button is silently hidden. Content browsing is unaffected.

すべてのページで以下の順番でスクリプトを読み込んでいます。順序を変えないでください。

All pages load scripts in the following order. Do not change the order.

<script src="js/data.js"></script>
<script src="js/script.js"></script>

運用上の注意 Important Notes

コンテンツの変更・削除について Modifying or removing content

原稿PDFが正本です。
コンテンツを変更・削除する場合は必ずPDFと照合してください。誤りを発見した場合は管理者に確認してから修正してください。
The PDF is the canonical source.
Always cross-reference the PDF when editing content. If you find an error, confirm with the administrator before correcting it.

検索エンジンへの非表示設定 Search engine exclusion

全ページに以下が設定されており、検索エンジンにインデックスされません。意図的な設定です。変更しないでください。

All pages include the following tag to prevent search engine indexing. This is intentional — do not remove it.

<meta name="robots" content="noindex,nofollow">

data.js の構文エラーに注意 Watch out for data.js syntax errors

data.js は JavaScript です。カンマの過不足・クォートの閉じ忘れがあるとページ全体が表示されなくなります。編集後はブラウザの開発者ツール（F12 → Console）でエラーがないか確認してください。
data.js is JavaScript. A missing or extra comma, or an unclosed quote will break all pages. After editing, open the browser's developer tools (F12 → Console) to check for errors.
